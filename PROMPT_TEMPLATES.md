# Cursor用プロンプトテンプレート集

## エラー修正時のプロンプトテンプレート

### 基本テンプレート
```
[エラーの指摘]
[具体的な問題箇所]で[問題の内容]が発生しています。

[期待する状態]
- [具体的にどうあるべきか]
- [どの要素に何を表示すべきか]

[禁止事項]
- [やってはいけないこと]
- [避けるべき変更]

[タスク]
既存構造を維持したまま、[具体的な修正内容]を、コード差分を最小にして修正してください。

修正後に、テキストの重複とレイアウト崩れが起きていないか、箇条書きで自己レビューを書いてください。
```

### テキスト重複修正の例
```
Heroセクションの見出しテキストが二重に表示されている。

期待する状態:
- 見出しは1つだけ、h1にのみ表示する
- 同じテキストを他の要素に複製しない

禁止事項:
- 新しいテキストノードを追加しない
- 同じコピーを他の要素に複製しない
- ファイル全体を書き換えない

タスク:
既存構造を維持したまま、重複を取り除き、コード差分を最小にして修正してください。

修正後に、テキストの重複とレイアウト崩れが起きていないか、箇条書きで自己レビューを書いてください。
```

### レイアウト崩れ修正の例
```
[セクション名]のレイアウトが崩れています。[具体的な問題: 例「要素が重なっている」「余白が不均一」]

期待する状態:
- 余白は8pxグリッドに揃える
- セクション内部の要素間は24px程度に統一
- セクションごとに上下80pxのマージン

禁止事項:
- 既存の正常なコードには触れない
- ファイル全体を書き換えない
- 変更箇所だけを編集する

タスク:
[具体的な修正箇所]のみを修正し、コード差分を最小にして修正してください。

修正後に、レイアウト崩れが起きていないか、箇条書きで自己レビューを書いてください。
```

## デザイン改善時のプロンプトテンプレート

### コントラスト改善
```
全体の本文カラーを#222、背景を#F5F7FA程度に変更し、WCAGのコントラスト比4.5:1以上を満たすように調整してください。

期待する状態:
- 本文テキスト: #222
- 見出し: #111
- 背景: #F5F7FA
- リード文や注釈: #444

禁止事項:
- 既存の正常なスタイルには触れない
- グラフの色は変更しない
- 変更箇所だけを編集する

タスク:
css/style.cssファイルの該当箇所のみを修正してください。

修正後に、コントラスト比が4.5:1以上になっているか確認してください。
```

### 情報密度改善
```
経済指標の解説セクションのカードについて、テキスト量を1カードあたり見出し + 要約 + 箇条書き6項目以内に収め、それ以外の文は削除してください。

期待する状態:
- 指標名 + 一行要約
- メリット（3点まで）
- デメリット（3点まで）

禁止事項:
- 新しいテキストを追加しない
- 既存の構造を壊さない
- ファイル全体を書き換えない

タスク:
frontend/js/main.jsのloadExplanations関数内のexplanationsオブジェクトのみを修正してください。
```

### 余白統一
```
セクションごとに上下80pxのマージンを取り、セクション内部の要素間は24px程度に統一して、情報のまとまりが分かるようにしてください。

期待する状態:
- セクション間: 上下80px
- セクション内部要素間: 24px
- 8pxグリッドシステムに揃える

禁止事項:
- 既存の正常な余白には触れない
- グラフコンテナのサイズは変更しない
- 変更箇所だけを編集する

タスク:
css/style.cssファイルの該当箇所のみを修正してください。
```
